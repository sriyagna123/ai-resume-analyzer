## AI Resume Analyzer
Live Demo
https://ai-resume-analyzersri.onrender.com
### Run locally

Open two terminals.

#### Backend

```bash
cd backend
npm install
npm start
```

Backend runs on `http://localhost:5000`.

#### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on `http://localhost:5173`.

### Notes
- Upload supports **PDF** and **DOCX** (not `.doc`).
- Create an account in the UI (Signup/Login) before analyzing.
- Do not commit `.env` or `backend/data/users.json` (already ignored).
